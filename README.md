# ML_Fuel_Analisys-Predictions
Questo progetto permette, una volta inserito un database, di predire il consumo di carburante in una flotta di veicoli aziendali tramite modelli di Machine Learning. L'obiettivo del progetto è ridurre la spesa del carburante, in modo da rendere le aziende di logistica e trasporti più efficienti.
